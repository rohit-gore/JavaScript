* JavaScript me hm 'var' keyword ko jyada use nhi krenge.
    var - > variable can be re-declared and updated.
* hm sirf 'let' aur 'const' ka hii use krenge.
    let -> variable cannot be re-declared but can be updated.
    const -> variable cannot be re-declared or updated.

* pair of {} defines a single block. iske bich me likha code usi block me chalta hai.
* {} block ke andr ek defined variable ko update nhi kr skte. sidha dusra block bnao.

*object -> ek collection of data or keys hota hai.
    objects ke andar keyss hoti hai.
    object ka datatype 'const' ho fir bhi keys ko change kra ja skta hai.

*arithmetic operators me 2 unary operators hote hai:
    1. Increment : ++
    2. Decrement : --

* 'prompt' me jb bhi koi input dalte hai .. vo hmesha string ke form me return krta hai.
    too == wala comparison operator use hoga === wala nhi.
    
* property vs methods:
    properties -> bss kuch value de deti hai.
    methods -> kuch kaam bhi krte hai.

* string are immutable, too hme pta hai vo change nhi hoti, aur hmesha sare operation nyi string pr hii hote hai.
    arrays me kuch methods usi array me change kr dete hai, kuch methods nayi array return krte hai.

* redundancy -> technical word which means repeation, code me redundancy ana accha nhi hai.

* JavaScript me hm ek function ko dusre function me pass kr skte hai as an argument
    uski ko hm callBack function bhi kehte hai
    isiiye JS me function special ko kaam ki chiz hai.

* Higher Order Function / methods :
    Ye vo functions hote hai jo dusre functions ko as a parameter le lete hai ya fir usko return krte hai.
    
* console.log --> print
  console.dir --> document --> properties, methods ko print krta hai.

* HTML ko JS ke andr access kr skte hai by:
    window -> document(model / DOM)

* Jb hm ek Object bnate hai too uske andr by default ek Object hota hai jisme
    sare methods hote hai(ex: push, pop, insert) is default object ko Prototype kehte hai.
    aur hmari jitni bhi Objects hoti JS me vo apne Prototypes se methods inherit krte hai.
